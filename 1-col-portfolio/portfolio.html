Aerospace Mission Control

Aerospace Mission Control is a planning, execution, and analysis tool for near-space aircraft launches.

AMC is built on pure HTML/JS/CSS on the front-end, and NodeJS + MongoDB on the back-end.

It is currently being used as a tracking and mission control tool for Penn Aerospace Club's High-Atmospheric Balloon launches.

Planned versions will include path forecasting using Monte Carlo simulations on meteorology data, and more in-depth analysis tools within the app.

Socratic

I initially designed Socratic to be the backend for my relational essays project, Socratic Network. In addition to having usual CMS capabilities, the "posts" in the app would be related to and weighed against one another. I want it to be a platform to discuss opposing ideas, and see what comes out.

Socratic is what powers that. A general-purpose CMS that can be modified and used as the backbone of any CRUD or REST application.

For example, Penn Aerospace Club's knowledge base is being built upon a modified version of Socratic. In addition, TSG is currently being built on Socratic as well.
